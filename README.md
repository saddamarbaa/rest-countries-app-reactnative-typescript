# Run_Locally

Clone the project

```bash
https://github.com/saddamarbaa/rest-countries-app-reactnative-typescript
```

Go to the project directory

```bash
 rest-countries-app-reactnative-typescript
```

Install dependencies

```bash
yarn install
```

Start the server

```bash
yarn start
```
